### Installing


Primeiro passo

Para instalar as dependências , execute o comando na raiz do projeto

```
composer install
```

Criar as tabelas no banco de dados 

```
php artisan migrate
```


Estartando nosso servidor

```
php artisan serve ou php artisan serve --port=8080
```


## Executando os testes
